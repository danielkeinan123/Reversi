Reversi Game:
המשחק מתחיל בצורה אוטומטית, כאשר בהפעלה ראשונית תור השחור.
כאשר מתחילים משחק חדש השחקן שהתחיל את המשחק מקבל את התור הראשון.

בכניעה לא ניתן יותר לשחק, ולחיצה על המשחק לא עובדת למעט על משחק חדש.

פונקציות מרכזיות:
clickCell: פונקציה הפועלת על לחיצה על תא - ומכאן מתחילים את הלוגיקה המרכזית של עדכון המסך והנתונים בהתאם לפעולה במשחק.
setCellsToNewColor: משנה את התאים לעיגול בצבע המתאים לפי הלחיצה.
createNewGame: יצירת משחק חדש.
updateStatistics: עדכון לוח הסטטיסטיקות
initStatistics: אתחול סטטיסטיקות בתחילת משחק חדש - מטרה של הפונקציה לאתחל את כלל המשתנים לערכם. הפונקציה מקבלת בנוסף פרמטר האם זהו המשחק הראשון.

בונוסים:
מימשנו את הבונוס של התחלה של משחק חדש.
